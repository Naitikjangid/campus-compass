# Campus Compass

A polished lost-and-found portal for a college campus, built with plain HTML, CSS, and JavaScript.

## Features

- Live search across item names, descriptions, colors, and locations
- Filters for report type, category, zone, timeline, status, and sorting
- Quick category chips for common student items
- Local report form that adds new listings instantly and saves them in `localStorage`
- Responsive, campus-themed interface with bold visuals

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy

This project is static, so it can be deployed by uploading the folder contents to:

- Vercel: create a new project and import/upload the folder as-is
- Netlify: drag and drop the folder contents into Netlify Drop
- Railway: deploy as a static site from the same files

The included `vercel.json` keeps URLs clean on Vercel.
